# TrackingBP - Blood Pressure Tracker

## Установка и запуск

1. Скачайте последний релиз: [TrackingBP.zip](https://github.com/DariAltpanika/TrackingBP/releases/latest)
2. Распакуйте в любую папку
3. Запустите `TrackingBP.vbs` (консоль не появится)

## Обновления

Запустите `update.bat` — откроет страницу GitHub с последней версией

## Требования

- Windows 10/11
- Не требует установки Java (всё уже внутри)
